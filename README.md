## 🚀 О проекте

Этот репозиторий содержит исчерпывающую документацию по **Tailwind CSS** — utility-first CSS фреймворку. Здесь собраны все необходимые классы, примеры использования, лучшие практики и стратегии создания адаптивных интерфейсов.

Главная цель — предоставить ресурс, которым можно пользоваться ежедневно:

- 🌐 **Онлайн-версия**: Удобный сайт с поиском и навигацией.
- 🖨️ **PDF-шпаргалка**: Готовый файл для печати с основными классами.
- 📚 **Структурированные гайды**: От установки до интеграции с фреймворками.

## 📖 Что внутри?

Документация разбита на логические блоки для удобного изучения:

### 🟢 Для новичков

- **Введение и установка**: Подключение через CDN, PostCSS, CLI.
- **Конфигурация**: `tailwind.config.js`, темы, кастомные значения.
- **Утилитарные классы**: Основные классы для стилизации.
- **Цвета и типографика**: Работа с палитрой и шрифтами.

### 🟡 Для уверенных пользователей

- **Flexbox и Grid**: Создание сложных макетов.
- **Responsive дизайн**: Breakpoints и мобильная адаптация.
- **Состояния**: Hover, focus, active, group-hover.
- **Dark mode**: Тёмная тема из коробки.

### 🔴 Для профи

- **Кастомизация**: Расширение конфигурации, кастомные утилиты.
- **Директивы**: `@apply`, `@layer`, `theme()`, `@screen`.
- **Плагины**: Создание и подключение плагинов.
- **Оптимизация**: PurgeCSS, tree-shaking, production build.
- **Интеграции**: React, Vue, Next.js, Svelte, Angular.

## 🛠 Как использовать этот репозиторий

### 1. Просмотр онлайн

Просто перейдите на сайт документации, созданный через **MkDocs**. Сайт автоматически обновляется при пуше в ветку `main`.

### 2. Локальная разработка документации

Если вы хотите дополнить гайд:

```bash
# Клонирование
git clone https://github.com/DaniilGavrin/tailwind-css-guide.git
cd tailwind-css-guide

# Установка зависимостей (требуется Python)
pip install -r requirements.txt

# Запуск локального сервера
mkdocs serve
```

Откройте `http://127.0.0.1:8000` в браузере.

### 3. Печать шпаргалки

Скачайте PDF из релизов или сгенерируйте его самостоятельно.
